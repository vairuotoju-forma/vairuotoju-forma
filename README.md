# Vairuotojų forma

Ši forma leidžia vairuotojams fiksuoti:

- savo vardą
- datą (automatinė)
- ridos pradžią
- tepalų lygį
- komentarus
- (neprivalomai) nuotrauką

Duomenys siunčiami į Google Sheets per tavo Apps Script nuorodą.

## Diegimas

1. `npm install`
2. `npm run dev`

Arba naudok [Vercel](https://vercel.com) automatiškai.
